## Featured Project

### 🧠 Parkinson's ON/OFF Medication Classification from EEG
> EEG-based binary classifier distinguishing Parkinson's patients on vs. off medication using a hybrid EEGNet-LSTM architecture.

**Key contributions:**
- Designed a **bidirectional LSTM with temporal attention** — replacing naive last-step readout with a learned weighted sum over all time steps
- Implemented a **leakage-free 3-way subject split** (inner-train / val / test) with explicit overlap assertions at every boundary
- Added **early stopping**, **ReduceLROnPlateau** scheduler, **gradient clipping**, and **Gaussian noise augmentation**
- Built a **grid search** across batch sizes and learning rates with per-config epoch history logging
